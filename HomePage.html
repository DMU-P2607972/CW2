Here are some Rules:

Conways Game of Life is a simulation of evolution which depends on how it started, what the initial amount of alive cells was and their positioning.

Dead cells look like this:


Alive cells look like this:


The rules are as follows:



  • if a living cell has more than 3 neighbour -> it dies

  • if a living cell has less than 3 neighbour -> it dies

  • if a dead cell has exactly 3 alive neighbours -> it comes alive

  • in other cases, the cell remains in its current state
